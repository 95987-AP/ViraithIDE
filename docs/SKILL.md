## Frontend Design Guidelines for VIRAITH IDE

### Theme: Caffeine Terminal

The design follows a terminal-inspired aesthetic with muted, warm colors:

#### Colors
- Background: `#0a0a0a` (near black)
- Surface: `#111111`
- Surface Elevated: `#1a1a1a`
- Border: `#262626`
- Text Primary: `#c9c9c9`
- Text Secondary: `#808080`
- Text Muted: `#5c5c5c`
- Accent: `#b8956b` (warm amber)
- Status Success: `#5a7d5a`
- Status Warning: `#8a7355`
- Status Error: `#7d4a4a`

#### Typography
- Primary: JetBrains Mono
- Fallback: SF Mono, Fira Code, monospace

#### Key Design Principles
1. No harsh or bright colors
2. Muted, terminal-like feel
3. Minimal UI chrome
4. Focus on content
5. Subtle animations
6. Clear visual hierarchy
